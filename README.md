# Fake_news_detector
An AI‑powered web app for detecting misinformation in news articles, with interactive UI, chat assist, and real‑time credibility scoring.
🚀 Features
Paste or URL‑based article input
International vs. National categories
Chatbot assistant for conversational fact‑checking
Real‑time credibility score, verdict & detailed breakdown
Responsive design, Tailwind CSS + React
🛠️ Tech Stack
Frontend: React, Tailwind CSS, Lucide icons
Backend: (if applicable) Flask/FastAPI or mock functions
ML: TF‑IDF + Logistic Regression, Decision Trees, etc.
Deployment: GitHub Pages / Netlify / Vercel
